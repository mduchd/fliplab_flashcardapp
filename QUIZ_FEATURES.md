### Joining
- **Join by Code**: Enter a specific Access Code to start a private quiz session.

### Exam Interface (Take Quiz)
- **Countdown Timer**: Visual timer that changes color (Green -> Orange -> Red) as time runs out.
- **Auto-Submission**: Quiz automatically submits when the timer hits zero.

- **Instant Feedback**: Immediate scoring upon completion (Score, Correct/Total count, Time spent).

- **Pass/Fail Status**: Clear indication of result based on the passing score.
- **Student History**: A log of all past quiz attempts and their outcomes.
